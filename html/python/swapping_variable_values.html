Swapping variable values¶

  • Author: Chris Albon, @ChrisAlbon
  • Date: -
  • Repo: Python 3 code snippets for data science
  • Note: Originally from: Learning Python

Setup the originally variables and their values¶

In [11]:
one = 1
two = 2

View the original variables¶

In [12]:
'one =', one, 'two =', two
Out[12]:
('one =', 1, 'two =', 2)

Swap the values¶

In [13]:
one, two = two, one

View the swapped values, notice how the values for each variable have changed¶

In [14]:
'one =', one, 'two =', two
Out[14]:
('one =', 2, 'two =', 1)
