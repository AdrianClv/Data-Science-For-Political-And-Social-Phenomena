### Apostrophe


```python
'Chris\'s book'
```




    "Chris's book"



###  New line


```python
print('Stay calm\nand\ncarry on')
```

    Stay calm
    and
    carry on


For raw strings (i.e. you don't want Python to parse escape characters) place use r''

###  Print exactly what is inside the quotation marks


```python
print(r'Chris\'s book')
```

    Chris\'s book
